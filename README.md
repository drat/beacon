# beacon
A web presence aggregator for people discovery

### Run
`python homing_beacon.py -h`

### Develop
`python -m pip install -r requirements.txt`

### Run Tests
`python -m unittest discover -v -s beacon/tests -p 'test_*.py'`

### Build Docs
`cd docs; make clean rst html`
